# FAHAD TV

**FAHAD TV** — An Android TV-style home launcher for Windows, built with Python and pywebview.

## 📖 About

FAHAD TV is a Windows desktop application that turns your PC screen (especially a mini PC connected to a TV) into an Android TV-style interface. The UI is rendered in a native window via the `pywebview` library, while a Python backend handles app logic: launching programs, managing shortcuts, browsing movies & shows (via TMDB), and following live football scores (via ESPN API).

## ✨ Features

- 🖥️ TV-style home screen with customizable tiles (add/edit/remove apps from the UI)
- 🎬 Entertainment section — trending movies & shows, search, and details
- ⚽ Sports section — live scores and fixtures for favorite football leagues
- 🎮 Dedicated rows for Apps & Entertainment, Quick Access, and Games
- 🎨 Themes, blurred backgrounds, and animations
- 📦 Packaged as a native Windows installer (built with cx_Freeze + Inno Setup)
- 📱 Optional Unified Remote integration for phone-as-remote control

## 📝 Summary

An open-source Windows app that mimics the Android TV experience: a customizable tile-based home screen for launching apps, programs, and games, designed for remote-control or TV navigation instead of the traditional Windows desktop.
